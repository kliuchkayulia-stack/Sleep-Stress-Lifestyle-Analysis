# Sleep-Stress-Lifestyle-Analysis
Exploratory data analysis of sleep quality, stress level and lifestyle factors across different professions using Python.
##  Опис проєкту
Цей pet-проєкт присвячений аналізу взаємозв’язку між якістю сну, рівнем стресу,
фізичною активністю та способом життя людей різних професій.

В умовах складного часу, повномасштабної війни, нічних тривог і постійного
психоемоційного навантаження тема сну та стресу є особливо актуальною.

##  Мета дослідження
- Проаналізувати, як якість та тривалість сну пов’язані зі стресом
- Порівняти показники між різними професіями
- Дослідити наявність розладів сну в різних групах
- Виявити можливі закономірності між фізичною активністю та рівнем стресу

## Використані інструменти
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

##  Основні кроки аналізу
- Очищення та підготовка даних
- Групування за професіями
- Кореляційний аналіз
- Візуалізація даних
- Формування висновків

## Основні висновки
- Вища якість сну пов’язана з нижчим рівнем стресу
- Залежність між фізичною активністю та стресом існує, але не є однаково
  вираженою для всіх професій
- Розлади сну та показники якості сну відрізняються залежно від професії
